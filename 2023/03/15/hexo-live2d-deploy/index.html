一、部署原理

在 Hexo + Butterfly 博客中部署 Live2D 前端组件，本质是通过引入基于 WebGL 的 JavaScript 库，在网页的 <canvas> 标签中解析并绘制模型文件。

Live2D 模型分为早期版本 Cubism 2.0 和现代版本 Cubism 3.0+，两者渲染管线差异显著。本教程使用 Cubism 3.0+ 的 .moc3 模型格式。核心依赖是 Live2d-Widget-v3 项目，它在 live2d-widget 的基础上接入了对 .moc3 文件的支持。

整个流程分三步：模型获取 → 引擎选择 → 网页部署。

二、模型获取

常用资源平台：

  • Live2D Cubism 官方资源库
  • Booth
  • Nizima
  • GitHub 搜索 live2d models moc3
  • B 站搜索「Live2D 模型配布」或「moc3 免费模型」

下载时确认资源包含 .moc3 文件，并注意版权许可。
