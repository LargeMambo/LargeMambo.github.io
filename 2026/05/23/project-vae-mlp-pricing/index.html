VAE + MLP 期权定价 — 深度学习 Greeks 预测框架

发表于2026-05-23|更新于2026-06-25|项目展示
|总字数:507|阅读时长:1分钟|浏览量:|评论数:

项目简介

这是一个将变分自编码器与 MLP 结合的期权定价深度学习框架。核心思路是用 VAE 对标普 500 期权波动率曲面进行特征压缩，再通过 MLP 解码出期权价格和四大风险指标，比传统 QuantLib PDE 方法加速 三个数量级。

本项目是 ljding94/VAE_pricing 的延续与扩展，新增了多头 Greeks 预测、亚式期权数据增强和完整的 3D 可视化系统。

核心结果

多头定价器（核心贡献）：

输出 测试 R² 单次推理 批量 1000
价格（NPV） 0.9988 < 1ms ~100ms
Delta 0.9983 — —
Vega 0.9905 — —
Gamma 0.9766 — —
Theta 0.9679 — —

对比 QuantLib PDE： QuantLib 单次 10–50ms，模型单次 < 1ms，加速超 1000×。

单头定价器（基线）：

产品 测试 R²
美式看跌期权 0.9935
亚式看涨期权 0.9954
亚式看跌期权 0.9977

模型架构

采用两阶段训练策略：先无监督预训练 VAE（压缩波动率曲面），再冻结编码器训练定价 MLP，最后联合微调。

数据流程

预训练模型

项目已提供全部预训练权重：

  • VAE 模型 276 KB：专业监控，820→10 维压缩
  • 多头定价器 195 KB：同时输出价格 + 4 个 Greeks
  • 单头定价器 各约 170 KB：美式/亚式期权专用

技术依赖

Python 3.8+，依赖 PyTorch、NumPy、QuantLib、SciPy 等开源库。
